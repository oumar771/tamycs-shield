# Tamycs Shield API Documentation

## Authentication Endpoints

### POST /api/auth/register
Register a new user

### POST /api/auth/login
Login with credentials

## Password Endpoints

### GET /api/passwords
Get all passwords for authenticated user

### POST /api/passwords
Create new password

### PUT /api/passwords/:id
Update password

### DELETE /api/passwords/:id
Delete password

### GET /api/passwords/search
Search passwords

## Category Endpoints

### GET /api/categories
Get all categories

### POST /api/categories
Create custom category
